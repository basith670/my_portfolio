## 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/basith670/my_portfolio.git
```

Move into the project

```bash
cd my_portfolio
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
